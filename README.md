# Dart Practice Problems

This repository contains beginner-friendly Dart programming problems designed to build a strong foundation in programming. These exercises cover basic concepts such as variables, constants, input/output, conditionals, loops, arithmetic operations, and string manipulation.

---

# Practice 1: Basic Dart Programs

### 1. Print Your Name

Write a Dart program to print your name.

### 2. Print Using Single and Double Quotes

Write a Dart program to print the following using both single and double quotes:

* Hello I am "John Doe"
* Hello I'am "John Doe"

### 3. Declare a Constant Integer

Declare a constant integer in Dart and assign the value 7 to it.

### 4. Calculate Simple Interest

Write a Dart program to calculate simple interest using the formula:
Simple Interest = (p × t × r) / 100

### 5. Square of a Number

Write a Dart program to calculate and print the square of a number using user input.

### 6. Print Full Name from First and Last Name

Write a Dart program to print the full name by taking first name and last name as user input.

### 7. Find Quotient and Remainder

Write a Dart program to find the quotient and remainder of two integers.

### 8. Swap Two Numbers

Write a Dart program to swap two numbers.

### 9. Remove Whitespaces from String

Write a Dart program to remove all whitespaces from a given string.

### 10. Convert String to Integer

Write a Dart program to convert a String into an integer.

### 11. Split Restaurant Bill

Write a Dart program to calculate the split amount of a restaurant bill among friends using the formula:
Split Amount = Total Bill Amount / Number of People

### 12. Calculate Time to Reach Office

Write a Dart program to calculate the time taken to reach the office using the formula:
Time = Distance / Speed

Given:

* Distance = 25 km
* Speed = 40 km per hour
  Print the result in minutes.

# Practice 2: Conditional Statements and Loops

### 1. Odd or Even

Write a Dart program to check whether a given number is odd or even.

### 2. Vowel or Consonant

Write a Dart program to check whether a given character is a vowel or a consonant.

### 3. Positive, Negative, or Zero

Write a Dart program to check whether a given number is positive, negative, or zero.

### 4. Print Name 100 Times

Write a Dart program to print your name 100 times.

### 5. Sum of Natural Numbers

Write a Dart program to calculate the sum of natural numbers up to a given number.

### 6. Multiplication Table of 5

Write a Dart program to generate and display the multiplication table of 5.

### 7. Multiplication Tables from 1 to 9

Write a Dart program to generate and display multiplication tables from 1 through 9.

### 8. Simple Calculator

Write a Dart program to create a simple calculator that performs:

* Addition
* Subtraction
* Multiplication
* Division

### 9. Print Numbers from 1 to 100 (Except 41)

Write a Dart program to print numbers from 1 to 100, but skip the number 41.
